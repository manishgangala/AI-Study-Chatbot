# 📚 AI Study Mentor Chatbot

An intelligent AI-powered Study Mentor built using Google Gemini API.

This project helps students learn concepts more effectively by providing beginner-friendly explanations, study guidance, roadmaps, motivation, and practical learning advice.

The project includes:

- 💻 Command Line Chatbot (Python)
- 🌐 Streamlit Web Application
- 🤖 Google Gemini AI Integration
- 🔐 Secure API Key Management using Environment Variables
- ⚡ Real-time Streaming Responses
- 📖 Educational & Structured AI Mentor Personality

---

## 🚀 Features

### AI Study Mentor
The chatbot is designed to:

- Explain concepts in simple language
- Help students understand difficult topics
- Create learning roadmaps
- Provide study strategies
- Answer coding and academic questions
- Encourage consistent learning habits

---

The web version provides:

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/yourusername/AI-Study-Mentor.git

cd AI-Study-Mentor
```

---

### 2. Create Virtual Environment

#### Windows

```bash
python -m venv venv

venv\Scripts\activate
```

#### Mac/Linux

```bash
python3 -m venv venv

source venv/bin/activate
```

---

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Setup Gemini API Key

Create a `.env` file in the project root:

```env
GEMINI_API_KEY=your_api_key_here
```

Get your Gemini API key from:

https://ai.google.dev/

---

## ▶️ Run Command Line Version

```bash
python main.py
```

Example:

```text
You: Explain Machine Learning

Chatbot:
Machine Learning is a way of teaching computers to learn patterns from data...
```

---

## 🌐 Run Streamlit Version

```bash
streamlit run app.py
```

Then open:

```text
http://localhost:8501
```
